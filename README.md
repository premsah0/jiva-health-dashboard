# Jiva Health Dashboard

A responsive healthcare admin dashboard built using Next.js 15, TypeScript, Tailwind CSS, and Zustand.

## Features

* User Management Dashboard
* User Detail Page
* Orders & Bookings
* Payment History
* Family Members Management
* Add User Modal
* Responsive Sidebar Navigation
* Dark / Light Theme
* Reusable Components
* Responsive Layout

## Tech Stack

* Next.js 15
* TypeScript
* Tailwind CSS
* Zustand
* Lucide React

## Getting Started

Install dependencies:

npm install

Run development server:

npm run dev

Open in browser:

http://localhost:3000

## Build Project

npm run build
